# Introduction
An auction system base on web3 implemeting with Next.js and solidity.

## Tech Stack
- Next.js(react tyepscript)
- solidity
- prisma
- tailwindcss
- antd
- etherjs

# Getting Started

```
pnpm install

npx prisma db push
npx prisma generate

pnpm run dev

```
